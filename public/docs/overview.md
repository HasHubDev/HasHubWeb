## API Endpoints

### Base URLs
```
Document API: https://api.hashub.ai/v1/doc/
Vector API:   https://api.hashub.ai/v1/vector/
```

### Authentication
All API requests require authentication using an API key in the Authorization header:
```
Authorization: Bearer YOUR_API_KEY
```
